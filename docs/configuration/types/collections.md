# Collections

## `List<E>`

```
[E, E, ...]
```

```json
[7, 9]
```

## `Map<K, V>`

```
{
  K: V,
  K: V,
  ...
}
```

```json
{
  "key1": "value1",
  "key2": "value2"
}
```

## `Multimap<K, V>`

```
{
  K: [V, V, ...],
  L: [V, V, ...],
  ...
}
```

```json
{
  "key1": ["value11", "value12"],
  "key2": ["value12", "value22"]
}
```

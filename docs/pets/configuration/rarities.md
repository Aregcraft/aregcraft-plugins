# `rarities/<name>.json`

| Name | Type | Description |
| --- | --- | --- |
| id | `String` | The identifier |
| name | `String` | The name, can have colors |
| level | `int` | The level |

```json
{
  "id": "RARE",
  "name": "%blue%%bold%RARE",
  "level": 1
}
```

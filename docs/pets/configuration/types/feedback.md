# `Feedback`

| Name | Type | Description |
| --- | --- | --- |
| sound | `PlayableSound` | The sound played |
| message | `String` | The message sent, can have colors |

```json
{
  "sound": {
    "type": "ITEM_ARMOR_EQUIP_LEATHER",
    "volume": 1,
    "pitch": 0
  },
  "message": "%name% %green%selected!"
}
```
